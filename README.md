# Docker Maltrail

Dockerized maltrail - Malicious traffic detection system

## Quick Start

```bash
docker run -d --name maltrail --net=host --privileged johackim/docker-maltrail
```

Point your browser to http://127.0.0.1:8338 (Password -> `admin:changeme!`)
